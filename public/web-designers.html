Our website translation work

We specialize in translating three kinds of websites:

  • Drupal sites - full integration with Drupal's i18n module that delivers effortless translation for Drupal 6 sites.
  • WordPress sites - our unique localization solution lets WordPress run multilingual. We can translate any WordPress blog or full sites.
  • Static HTML websites - full automation for turning a static HTML site, build with dozens of files into a multilingual website.
