To start sending translation jobs, you'll first need to create an account. Creating an account is free. You'll only be asked to pay when a translator starts working for you. You'll be able to:

  • Set up website translation projects, get exact prices and chat with translators.
  • Hire a translator for any language work you need.
  • Post instant text translation projects.
  • Create a free support center for your website, allowing you to communicate with your visitors using support tickets.
  • Set up instant interpretation for your visitor contacts.
