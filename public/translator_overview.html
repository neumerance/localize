The easy way to translate complete websites

Overview

HTML is the language used to code websites. Web browsers know this language and can extract text and styling information.

This is an example HTML code, taken from this very page:

<h2>Overview</h2>
<p><acronym title="HyperText Markup Language -
a language used to code web pages.">HTML</acronym>
is the language used to code websites.
<acronym title="Internet Explorer, Firefox, Opera and Safari are web browsers.
These programs can display web pages">
Web browsers</acronym>
know this language and can extract text and styling information.</p>

With a little effort, you can see what's displayed with this code. But, it's clear that extracting the text from a complete website, translating it and then putting it all back to the original HTML format, is going to take a lot of work.

And, what's going to happen when the website changes and you need to update the translation?

Automating the technical work

Using Translation Assistant, you don't need to edit HTML code at all.

The software goes through the entire website and extracts all texts from it. Then, it lets you edit the website using a built-in browser/editor. This means, you work on the web pages as they look and just translate one sentence at a time.

When you're done, the entire website, including the translated pages, is built. The translated website is automatically built by Translation Assistant on the client's computer using your translation. When you're done translating, the job is complete. Period.

Features

Translation Assistant comes with all the tools you need to get the job done fast

  • Revisions Management - new or modified texts are detected
  • Translation Memory - you don't need to translate anything more than once. Any sentence you've already translated can be automatically used.
  • Built-in WYSIWYG editor. No need for any other software.

How does it work?

Sign up and set up your account

You need to sign up for a translator account. It's free. There's no sign-up, subscription or any other hidden fee.

Your profile will include your resume, languages you can translate in and fields of expertise. The systems will also create a small practice project for you, to familiarize with the software.

Find projects and get to work

Once you've completed your practice project, you can start working on live projects. The system locates projects that match your profile.

You can communicate with the client, to better understand any special requirements and make your bid.

When your bid is selected, the project is downloaded to your PC. At this time, the client deposits the entire project payment with us, where we hold it for you.

Get paid

Your payment is released when the work is done. We take a 20% commission and you get the rest.
