Step three - building the translated website

New HTML files are created

The translated website is created automatically. Translation Assistant can generate the translated website in a number of ways:

  • All languages placed in a single domain
  • Each language goes to a different domain
  • Each language goes to a subdirectories in the same domain

In addition, you can choose and customize the way language selectors are placed, to move from one language to the other.

Your manual edits are saved

The translation software keeps track of any edits you do on the translated website and retains these edits.

Any edits that you make to the translated pages are recorded. Next time Translation Assistant creates the translated pages, you'll find all your edits are still there.
