Step one - extracting texts from the website

HTML scan

Translation Assistant scans your website. It extracts all texts, including:

  • All normal text
  • Meta information
  • Image ALT tags
  • Titles, acronyms, etc.
  • Texts from Javascript (for menus, etc.)

You can extract texts from either HTML files on your PC or from the web.

Only new or modified texts are marked for translation

If this is the first time your website is translated, all texts are extracted. Otherwise, only new or modified texts are marked for translation.

When the program counts words, repeated texts are automatically ignored. You only pay for work that the translator really needs to do.
